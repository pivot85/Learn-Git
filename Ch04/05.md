## git commit الامر

Committed الى مرحلة Staged هو امر يقوم بنقل الملفات من مرحلة 

`git commit`

في حال اردنا اضافة رسالة توضيحية للملفات التي سننقلها فنقوم بكتابة الاتي

`-m`

message وتعني 

ويكون الامر كالاتي

`git commit -m "Example"`
